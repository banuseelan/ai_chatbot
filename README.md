# AI Chat bot

A Python chatbot that is not connects to GroqCloud AI to answer any question.
Built during Week 2 of my 5-week tech learning journey.

## What it does
- Greets the user by name
- Saves the full conversation to chat_history.txt
- Handles errors gracefully so it never crashes
- Exits politely when user types 'quit'

## Tools used
-Python (all built-in libraries - no installation needed)
-GroqCloud API (free tier)
-online-pthon.com (browser-based Python editor)

## What i learned
-Python variables, functions, loops, if/else logic
-How APIs work and how to connect Python to external services (But it's not connecting)
-How to handle errors in real programs
-How to save data to files
-How to upload and document code on GitHub
